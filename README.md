# Service Marketplace Web Application

A full-stack web application built using **React and Django**, designed to connect customers with nearby service providers such as plumbers, electricians, and carpenters.

The platform simplifies the full service workflow, allowing users to discover local providers, book appointments, complete payments, receive invoices, and leave ratings â€” all through a unified web system.

## Key Features
- User authentication and role-based access (customers & service providers)
- Location-based discovery of nearby service providers
- Service categorization (plumbing, electrical, carpentry, etc.)
- Appointment booking with flexible scheduling
- Secure payment workflow
- Automatic invoice generation
- Rating and review system
- RESTful APIs built with Django
- React frontend consuming backend APIs
- Relational database design and integration

## Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Django, Django REST Framework
- **Database:** PostgreSQL 
- **Tools:** Git, GitHub, Linux

## Installation & Run
1. Clone the repository  
2. Create a virtual environment and install dependencies  
3. Run Django backend server  
4. Start React frontend  

## Project Goal
To demonstrate strong full-stack development skills using React and Django, focusing on backend logic, API design, payment flows, and scalable SaaS-style architecture.
