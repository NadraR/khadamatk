اختبار صفحة تفاصيل الفاتورة

معلومات الاختبار

هذه الصفحة لاختبار صفحة تفاصيل الفاتورة في تطبيق خدماتك

  • تأكد من تشغيل الخادم الخلفي (Backend) على المنفذ 8000
  • تأكد من تشغيل التطبيق الأمامي (Frontend) على المنفذ 3000
  • تأكد من وجود بيانات فاتورة في قاعدة البيانات

روابط الاختبار

الصفحة الرئيسية صفحة العميل اختبار فاتورة رقم 1 اختبار فاتورة رقم 2 API - جميع الفواتير API - فواتيري
أوامر التشغيل

الخادم الخلفي:

cd backend && python manage.py runserver

التطبيق الأمامي:

cd frontend && npm start
